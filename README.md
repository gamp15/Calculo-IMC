# Calculadora de IMC

Projeto simples em HTML, CSS e JavaScript que calcula e classifica o IMC do usuário.

## Funcionalidades
- Entrada de peso e altura
- Cálculo automático do IMC
- Classificação (baixo peso, normal, sobrepeso, obesidade)
- Interface simples e responsiva

## Tecnologias
- HTML
- CSS
- JavaScript
